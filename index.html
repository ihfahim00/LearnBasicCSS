There are 5 types of selectors. This page will explain the CSS simple selectors.

Here I used "Tag" selector for using Inline css.

Here I used "Tag" selector for using Internal css. The element selector selects HTML elements based on the element name.

Here I used "Class" selector for using External css. The class selector selects HTML elements with a specific class attribute. To select elements with a specific class, write a period (.) character, followed by the class name.

Here I used "ID" selector for using External css. The id selector uses the id attribute of an HTML element to select a specific element. The id of an element is unique within a page, so the id selector is used to select one unique element! To select an element with a specific id, write a hash (#) character, followed by the id of the element.
